# 📊 Airbnb Data Analysis (Power BI Project)

## 📌 Overview
This project analyzes Airbnb dataset to understand pricing trends, booking patterns, and location-based performance. The goal is to derive insights that can help improve business decisions and maximize revenue.

## 🛠 Tools Used
- Power BI  
- Microsoft Excel  

## 🔍 Key Insights
- Significant price variation across different cities and locations  
- High-demand cities identified (e.g., Paris, New York, Sydney)  
- Relationship between availability and pricing analyzed  
- Ratings and room types impact pricing and demand  

## 📈 Dashboard Features
- KPI Cards (Average Price, Total Listings, Ratings)  
- City-wise market share analysis  
- Room type pricing comparison  
- Ratings distribution  
- Interactive filters for better analysis
